# config-Ubuntu
configuration de ubuntu pour une utilisation comme windows

## installation des outils de base
```bash
sudo apt install htop lnav 
```

## capture d'écran
installer flameshot
```bash
sudo apt install flameshot
```
dans paramètres > Devices → Keyboard → Shortcuts.
désactivé capture d'écran (backspace)
ajouter un raccourcis custom > commande : ```flameshot gui```
``` fn + inser (impr écran)```

## papier presse
```bash
sudo apt insatll gnome-tweaks gpast
```
ouvrir menu gpast > Raccourcis clavier > supprimer les raccourcis
dans lancer l'interface graphique ```<Ctrl><Alt>v```   fermé
Ctrl + Alt + v       -> ouvre l'historique   -> clique > et colle ou tu veux
